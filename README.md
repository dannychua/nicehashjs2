The javascript library for Nicehash API v2.

![alt text](./nicehashjs.jpg "Logo Title Text 1")

## Installation

    npm install --save nicehashjs2

## Usage

    const NicehashJS = require('nicehashjs2');
    const nhClient = new NicehashJS({
        apiKey: <myApiKey>, 
        apiSecret: <myApiSecret>
    });
